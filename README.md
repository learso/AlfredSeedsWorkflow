# 简介
利用Alfred对Seeds进行操作，首次使用请配置自己的API。

# 功能
1. 快速添加文本笔记到Seeds，快捷键sa
![seeds1](https://user-images.githubusercontent.com/960238/123955161-e4863d00-d9db-11eb-962e-aaf97377797c.PNG)

2. 快速添加文本笔记到Seeds，带标签选择，快捷键sat
![Xnip2021-07-06_08-44-25](https://user-images.githubusercontent.com/960238/124526947-c9ca2300-de36-11eb-9823-1f14848f30de.jpg)
标签请在环境变量里设置，每个标签之间用半角逗号“,”分隔
![Xnip2021-07-06_08-39-39](https://user-images.githubusercontent.com/960238/124526983-e9f9e200-de36-11eb-95b4-98fd29a7a601.jpg)

3. 通过File Action上传图片
